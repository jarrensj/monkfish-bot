# monkfish-bot

Telegram bot frontend for toggling koi-fish algos and managing allocations.

## Quick Start
1. `cp .env.example .env` and set `TELEGRAM_BOT_TOKEN`.
2. `npm run dev` (or `npm run start` in prod).
